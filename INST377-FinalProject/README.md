# INST377-FinalProject: Scheduling Navigator

## Project Description
This is a web application designed to solve scheduling and meeting coordination issues by integrating with Google Calendar, Nager.Date for holidays, and a custom Supabase database for user preferences. The core features include finding free time slots across multiple attendees and managing user-specific settings.

## Target Browsers
Optimized for modern desktop browsers (Chrome, Firefox, Safari) and contemporary mobile browsers (iOS Safari, Android Chrome).
