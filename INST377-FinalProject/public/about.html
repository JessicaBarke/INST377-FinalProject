About the INST377 Scheduling Navigator

This application addresses the problem of **efficiently managing schedules and coordinating meetings** for students, professionals, and organizations.

By integrating with the Google Calendar API, Nager.Date, and a custom Supabase backend, we provide a streamlined way to find optimal meeting times, avoiding conflicts and holidays.

Key APIs Leveraged:

  • **Google Calendar API:** For real-time event and availability data (`Events.list`, `FreeBusy.query`).
  • **Supabase Database (Custom API):** To save and retrieve user preferences and default settings.
  • **Nager.Date:** To filter out public holidays from availability calculations.
